# Description of the main screens

## WELCOME TO JEERAAN SPLASH:

- Welcome and Onboarding Screen: Introduces the app and its purpose to new users.

## JOIN or REQUEST TO JOIN NEIGHBORHOOD:

- Join or Create Neighborhood screeb: This screen allows users to either join an existing neighborhood using an invitation code or request to join a neighborhood.

## HOME / MAIN DASHBOARD:

- Home screen: This is the main dashboard, offering quick access to announcements, members, invites, events, voting, service orders, forum, classified ads, grievances, and recent activities.

## ANNOUNCEMENTS:

- Create New Announcement screen: This screen enables users to compose and post new announcements, with options for text and image attachments.

- Recent Announcements screen: This screen displays a list of recent announcements for the current month, featuring a search bar, category filters, and a clear list of neighborhood updates.

## MEMBERS

- Members List & Invites: A new screen where residents can see who's online, send direct messages, and request invitations for new neighbors.

## INVITES

- Send Invite Requestscreen: This screen features the name, phone number, and optional email fields, as well as the note about administrative approval. You can now see it on the canvas.

## EVENTS

- Create New Event: This screen provides a form to create new events, including fields for title, details, organizer, venue, date/time, status (Upcoming, Past, Cancelled, Ongoing), and notes.

- Event List: This screen displays a list of all events, with each event showing its title, date, time, and organizer, along with filtering options.

- Events Calendar: This screen presents a monthly calendar view where days with events are highlighted, and clicking on a day reveals its scheduled events.

## VOTING

- Board Election Ballot: This screen lists the candidates running for the Board, allowing members to cast their ballots.

- Candidate Profile: This screen displays a candidate's bio, contact info, and messaging functionality.

- Candidate Q&A Screen: This screen provides a focused view of all neighborhood inquiries and the candidate's responses, making the election process more transparent.

- Candidate Profile Submission: This screen provides a form for candidates to submit their profiles, including a picture upload, bio, and a text box for their assets and community contributions.

- Election Information: this screen houses all election-related details.

- Vote Confirmation: This screen confirms a successful vote cast on the Board Election Ballot screen, showing the selected candidate and options to view results or return.

## SERVICE ORDERS:

- Submit Service Order screen: This screen provides a form for residents to submit new maintenance issues, including fields for unit address, occupant name, dates, optional maintenance person selection, issue details, and a satisfaction rating.

- Recent Service Orders screen: This screen displays a list of recent maintenance requests for the current month, featuring a search bar for filtering by unit number or occupant name, and filters for year/month and satisfaction rating.

## FORUM:

- Neighborhood Forum screen: This screen displays discussion topics, allowing users to view, start, and reply to threads.

## CLASSIFIED ADS

- Classified Ads screen: This screen presents classified ads from neighbors, with options to view details or post new listings.

- Classified Ads Create New screen: This screen has been generated for users to create and submit new classified ads, featuring fields for ad title, price, description, contact information, and an option to upload an image.

## Grievances

- Submit Grievance Form: A new screen where users can fill out a title, description, and upload images to present their case.

- Grievances List: A central place for members to see all active grievances, including who posted them, when, and how many comments they've received.

- Grievance Details: A detailed view for each grievance, complete with a comment section to foster neighborhood communication.

## RECENT ACTIVITIES

- Recent Activities List: This screen will display a chronological feed of recent neighborhood activities, including new announcements, completed service orders, forum posts, and upcoming events.

## ADMIN ANALYTICS:

- Admin Analytics Dashboard: This screen provides a UI mockup for the admin, displaying key metrics like active users, new sign-ups, announcement posts, work orders, voting participation, and classified ad activity, using charts and summary cards for data visualization. "Pending Invite Requests" section is added to the dashboard so the administrator can easily approve or decline new membership requests.
